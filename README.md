# Property Management System

A PHP-based property management application with multi-role user support.

## Features

- Multi-role user system (Property Owner, Tenant, etc.)
- API-based architecture
- Easy installation with database setup
- User registration and authentication
- Role-based access control

## Installation

1. Upload files to your LAMP server
2. Navigate to `/install` in your browser
3. Follow the installation wizard
4. Complete database setup and configuration

## Requirements

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache with mod_rewrite enabled
- PDO MySQL extension

## Project Structure

```
/
├── api/              # API endpoints
├── config/           # Configuration files
├── database/         # Database scripts
├── install/          # Installation wizard
├── public/           # Public web root
├── assets/           # CSS, JS, images
└── includes/         # Shared PHP files
```
